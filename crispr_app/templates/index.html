Base Editing Studio (ABE / CBE)

window efficiency · bystander · purity

For each designed guide, score base editing in the activity window (positions 4–8): the window efficiency of the target base (peak ~position 6), any bystander bases the editor would also hit, the resulting editing purity, and a composite BE score = efficiency × purity (higher = cleaner, more efficient single edit).

Prime Editing Studio

pegRNA · Spacer + RTT + PBS

Design pegRNAs for single-base substitutions, ranked by PRIDICT2.0-informed determinants (PBS Tm ≈ 37 °C, RTT/PBS length, 3′ homology) plus a lightweight 3′-extension secondary-structure penalty (a self-folding extension can't prime — shown as Fold).

Multiplex / Pooled Library Designer

greedy marginal-gain · diverse set

Pick a small set of guides for the current target that are individually strong and collectively diverse — a greedy optimiser adds, at each step, the guide with the best marginal gain = score − diversity × similarity-to-the-set. Diverse sets avoid the repetitive sequences that drive recombination in pooled libraries. Uses the Editing goal and Ranking chosen above.
